Consea

Search ABAP constants in classes and interfaces the googleish way

View project on GitHub

Welcome to ConSea home.

ConSea lets you search and find public constants that are defined in ABAP classes and interfaces and use them in your coding. ConSea features: *regular expression to search for constants *use of shared objects to speed up the inital loading of constants *search view plug-in for Eclipse-based ABAP Development Tools
